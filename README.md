## Dependencies

```
make install-dependencies
```

## Usage

```
./metro 圓山 忠孝復興
```

## API

```
>>> from rapid.systems import TaipeiMetro
>>> metro = TaipeiMetro()
>>> metro.show_map()
>>> metro.search_routes('圓山', '忠孝復興', 1)
```
